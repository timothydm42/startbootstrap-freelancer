Placid Discussions

My idea here is to create a debate type forum that uses natrual language processors to moderate each submission. I leveraged the IBM Watson Tone Analyzer and Natural Language Processor APIs through Bluemix. The app will indicate inadmissability down to the specific sentence, as well as vet the linked source material so that users can quickly guage the uitility of each post.

The app is built with Angular 1, Node, PostgreSQL on an Amazon remote database, and a few NPM packages including Express, Massive, and Gulp

My Inventory

You know, just another boring inventory maintenance app...just kidding! This app is integrated with the Amazon Alexa. Manual inventory upkeep is tedious and frequently error prone, because many businesses like restaurants use their stock throughout the day, and then have to recount it later or try to rember what they used, and it's a drag. Telling Alexa to update your database for you as you exit the store room saves a lot of time and tedium.

I came up with the concept and recruited talented developers Nathan Smith and Mikhail Collins and together we built the app using Create-React-App, Node, and PosgreSQL, with Axios, Express, Massive, Socket.io, Webpack and the Amazon Alexa-sdk and database services.

My specific work is mostly in the UI logic and communication loop; directing the data from client to server to database etc.
